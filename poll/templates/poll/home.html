FairVote

Blockchain as a Solution

Wouldn't it be nice to be allowed to vote using a secure online platform that provides transparency into elections results? Imagine an online system where it is possible to vote using a computer, tablet, mobile phone or a multimedia totem without the need for voters to physically access the polling station, yet offering the same or better guarantees compared to a public election:

  • Secure and anonymous votes, which can be verified at any moment
  • Impossibility to vote twice or to commit electoral fraud
